# Trip - Creation

Creating a trip is simple: just enter its name. Optionally, you can set a currency (defaulting to your user settings if left blank), add an image, and specify a date range.

<img src="/trip/img/trip_create.png" alt="Trip creation modal" />
<div style={{textAlign: 'center'}}><sup>Trip creation modal</sup></div>

Once created, your new trip will appear in your trips list.

:::tip
The optional date range automatically generates the trip days, labeled as `dd mmm.`, with corresponding dates set for you.
:::
