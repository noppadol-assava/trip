# Trip - Places

:::note[TL;DR]
A trip references places. Manage the referenced places and create new ones from the places panel.
:::

A trip can reference your places and those of invited members, to associate them to the plans.

:::info[important]
Creating a place from the trip interface is a shortcut, the place itself will not be owned by the trip.
:::

The referenced places are displayed in the _Places_ panel.
